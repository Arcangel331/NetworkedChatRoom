	Private Message
	Example: @Username Hello
	This will send a private message to a specific user

Instructions:
ChatClient can be run by typing "java ChatClient <host> <portNumber>"
ChatServer can be run by typing "java ChatServer <portNumber>"

Client:
Messages can be sent without using the command "SHOUTOUT",
For example "Hello", instead of "SHOUTOUT Hello". Because
the user should not have write SHOUTOUT everytime they want 
to send a message 

The user can exit by typing the command "EXIT" to leave the chatroom

Server:
All users will be notifed if someone joins the chatroom or leaves the 
chatroom. Also, if a user joins they will be greeted individually by the
server.
